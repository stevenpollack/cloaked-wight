# cloaked-wight
Repo for my various profile and config files
___

Dependencies:

* (`sudo apt get`) tmux
* (`sudo apt get`) vim
  * vundle
  * you'll need `gvim` -- (`sudo apt-cache search gvim`)
    -- for `+clientserver` vim capabilities
* (`sudo apt get`) zsh
  * oh-my-zsh

_Note:_ `tmux/.tmux.conf` is based on pivotal's tmux config
